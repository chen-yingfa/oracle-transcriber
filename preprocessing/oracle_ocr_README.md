# 单字的命名格式

![img_1.png](img_ocr_readme_0.png)

甲骨文字编（上）_cut_001_2_0_1_人_20346(A1).png

以'_'为分割

* 甲骨文字编（上）_cut_001 -> 为甲骨文字编（上）书中字形表的第n页
* 2 -> 为该页中第2行(这里的行数不一定准确, 但应该能大致定位到)
* 0 -> 为该行中从左到右第0个字形
* 1 -> 该字在书中的编号
* 人 -> 识别出来的汉字
* 20346 -> 对应为识别出来的汉达甲片编号 H20346
* A1 -> 字形时代代号

* 其余'oracle_*'文件夹下的命名规则与以上一致, 基本是以上规则的某一部分
